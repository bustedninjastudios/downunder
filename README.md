# Down Under

A short 2D Motherload-inspired game.

## Build Instructions
For itch.io, select HTML5 as your export target. Export it to a folder, rename "down under.html" to index.html, zip and upload.

## Credits
Soundtrack is from Open Game Art
 
## AI usage
AI was used for debugging and otherwise menial labour. 
Usage:
- player.gd: Mining checks and the raycasting
- shop_ui.gd: Shop items
No game assets were made with AI. Programmer art all the way 
 
Made for Campfire 2026 and Ember 2026.
